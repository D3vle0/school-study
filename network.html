네트워크

네트워크의 구성요소

  • 정보 기기 (컴퓨터, 프린터, 스캐너)
  • 송수신 장치 (LAN카드, 모뎀)
  • 전송 매체 (꼬임선, 동축 케이블, 광섬유 케이블)

네트워크의 구성 방식

  • 스타형

    • 중앙에 있는 허브를 중심으로 다른 컴퓨터들을 직접 연결
    • 허브가 고장나면 네트워크 마비

  • 링형

    • 이웃한 컴퓨터를 서로 연결하여 고리 형태로 구성 → 양방향 정보전송
    • 일부 회선에 문제가 있어도 정보 전송 가능하지만, 하나의 컴퓨터가 고장나면 전체 네트워크가 동작 X

  • 그물형

    • 컴퓨터들을 직접 회선으로 연결
    • 일부 회선에 문제가 있어도 다른 경로를 통해 정보전송
    • 광대역 통신망에 사용, 회선 설치 비용 많이 듦

  • 버스형

    • 하나의 전송 회선에 여러 대의 컴퓨터가 연결
    • 저렴하지만 관리 어렵, 전송 대기상태 자주 발생

  • 트리형

    • 중앙에 있는 하나의 컴퓨터에 일정 지역의 컴퓨터까지 하나의 회선으로 연결, 지역의 컴퓨터에는 여러대의 컴퓨터 연결
    • 쉽게 확장 가능, 중앙에 장애가 발생하면 전체 네트워크에 영향을 줌

OSI 7 Layer

  • Layer 7 (응용) - 사용자가 응용 프로세스에 접근할 수 있도록 해줌
  • Layer 6 (표현) - 입력 또는 출력되는 데이터를 하나의 표현 형태로 변환
  • Layer 5 (세션) - 통신 장치간의 상호작용 설정, 유지, 동기화
  • Layer 4 (전송) - 정보를 전송할때 발신지 대 목적지 간에 제어와 에러를 관리
  • Layer 3 (네트워크) - 정보를 발신지로부터 목적지로 전달할 책임을 갖는 계층
  • Layer 2 (데이터 링크) - 오류없이 한 장치에서 다른 장치로 정보 전달
  • Layer 1 (물리) - 물리적 매체로 정보전달

데이터 송수신의 동작 원리

  • 단방향 통신
    • 하나의 통신 회선으로 두 장치를 연결해 한쪽 방향으로만 정보를 전송할 수 있음
      EX) 라디오, 공중파 TV
  • 반이중 통신
    • 하나의 통신 회선으로 두 장치를 연결해 양쪽 방향으로 일차선 방식으로 전송할 수 있음
      ⇒ 한 쪽에서 정보를 보내면 한 쪽에서는 받을 수만 있음
      ex) 무전기, 팩스
  • 전이중통신
    • 두 개의 통신 회선으로 두 장치를 연결해 양쪽 방향으로 동시에 정보를 주고 받을 수 있음
      ex) 전화기, 컴퓨터 통신(OSI 7 Layer 사용)
  • 직렬 전송
    • 하나의 통신 회선을 이용해 한번에 한 비트씩 차례로 전송함
      ⇒ 전송 속도가 느리지만 회선 설치비용을 적게 들여 경제적임
  • 병렬 전송
    • 여러 개의 통신회선을 이용해 동시에 여러 비트씩 전송함
      ⇒ 전송 속도가 빠르지만 회선 설치비용이 많이 듬
  • 동기 전송
    • 송신측과 수신측이 하나의 기준 시간에 맞추어 동기화된 데이터를 주고 받음
      EX) 고속 통신망
  • 비동기 전송
    • 송신측이 데이터를 전송하면 수신측은 시간적 위치에 관계없이 비동기화 된 데이터를 수신함
      EX) 컴퓨터 통신(OSI 7 Layer 사용)

프로토콜

  • TCP/IP : 인터넷 표준 프로토콜
  • HTTP : 웹 상에서 정보를 주고받을 때 이용하는 프로토콜
  • SMTP : 메일 보낼 때 이용하는 프로토콜
  • POP3 : 메일 받을 때 이용하는 프로토콜
  • FTP : 파일 전송 프로토콜
  • TELNET : 원격지 컴퓨터 시스템에 로그인 할 수 있도록 도와주는 프로토콜
  • DHCP : 클라이언트가 동적인 IP 주소를 할당받아 인터넷을 사용 할 수 있게 해주는 프로토콜

IP 주소와 클래스

IPv4: 32비트

IPv6: 128비트

DNS: 각 사이트의 도메인 정보와 IP 정보를 데이터베이스에 보관하고 있다가 해당 사이트 접속을 원하는 신호가 오면 이를 번역해 해당 IP를 가진 사이트의 호스트 서버에 연결시켜주는 서버

도메인 네임

IP 주소는 숫자로 구성되어 있어 사용자가 이용하는데 어려움이 있다.

도메인 네임: IP 주소에서 사용하는 숫자를 문자로 바꾸어 우리가 쉽게 알 수 있도록 나타내는 것

DNS에 의해 IP 주소를 자동으로 바꾸어 줌

krnic.kisa.or.kr

  1. krnic : 호스트 컴퓨터 이름
  2. kisa : 소속 기관 이름
  3. or : 소속 기관 종류
  4. kr : 국가 도메인

comsi.devleo.us

  1. comsi : 호스트 컴퓨터 이름
  2. devleo : 소속 기관 이름
  3. us : 국가 도메인

국제 도메인과 국내 도메인

국가 분류 도메인 비고
국제 com 영리 법인
국제 org 비영리 법인
국제 gov 국가 기관
국제 info 정보 제공
국내 co 영리 법인
국내 or 비영리 법인
국내 go 국가 기관
국내 pe 개인

인터넷 서비스

  • 월드 와이드 웹 (WWW : World Wide Web)
    • 인터넷에서 웹 페이지들의 정보를 쉽게 주고받을 수 있도록 하는 서비스
    • HTTP 프로토콜 사용
    • HTML 문서와 이들 문서들을 서로 연계시켜 구성하는 웹페이지들의 집합체
    • 소리, 그래픽, 동영상 등의 정보를 하이퍼링크 구조로 연결해 다양한 멀티미디어 정보 제공
  • 사물인터넷 (IoT : Internet of Things)
    • 센서와 통신 기능이 있는 기기들을 인터넷으로 연결해
      사람의 개입 없이 수집한 정보를 서로 주고받아 스스로 일을 처리하는 기술
      ⇒ 취약한 보안, 사생활 침해 등 문제 발생
  • 클라우드 서비스 (Cloud Service)
    • 인터넷과 연결된 서버에 데이터나 소프트웨어를 저장해 놓고
      필요할 때마다 정보 기기로 접속하여 사용할 수 있는 서비스
  • 소셜 네트워크 서비스 (SNS : Social Network Service)
    • 온라인 상에서 특정한 관심이나 활동을 공유하는 이용자끼리
      사회적 관계를 맺고 대화나 정보를 교류하는 서비스
      ⇒ 저작권 침해, 개인에 대한 인권 침해 등 문제 발생
